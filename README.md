# ChessBotML

A machine learning chess bot that learns a player's style from historical games and predicts moves from board positions.

## Features
- Chess.com game extraction
- FEN generation
- Stockfish-assisted evaluation
- Style learning
- React web interface
- Play as White or Black

## Tech Stack
- Python
- Scikit-learn
- Stockfish
- React
- Vite
- chess.js
- react-chessboard

## Installation

### Backend
pip install -r requirements.txt

### Frontend
cd chess-bot-web
npm install
npm run dev

### Notes
stockfish is not included, you can download it from the website https://stockfishchess.org/download/ and add stockfish.exe to /server folder
